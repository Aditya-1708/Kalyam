# 💊 Kalyam Pharmaceutical

![Kalyam Logo](public/images/logo.png)

A professional, modern, and high-performance corporate website for **Kalyam Pharmaceutical**. This platform showcases human and veterinary pharmaceutical solutions, company values, global reach, and career opportunities.

---

## ✨ Features

- 🚀 **Modern UI/UX**: Built with React and Tailwind CSS for a sleek, premium feel.
- 📱 **Fully Responsive**: Optimized for all devices—desktops, tablets, and mobile.
- 🏢 **Corporate Sections**:
  - **About Us**: Detailed company history and vision.
  - **Leadership**: Insight into the team driving innovation.
  - **Global Presence**: Interactive showcase of global reach.
  - **Timeline**: A visual journey of company milestones.
- 📦 **Product Showcase**: Categorized view of human and veterinary healthcare products.
- 📄 **Careers Portal**: Integrated application system for prospective talent.
- 📞 **Smart Communication**:
  - Functional Contact Form.
  - One-click WhatsApp integration.
  - Click-to-call and Email functionality.
- 🎭 **Animations**: Smooth scroll reveals and micro-interactions using custom reveal components.
- 📜 **Certifications**: Trust-building section highlighting industry standards.

---

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) (Vite)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Routing**: [React Router](https://reactrouter.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Animations**: CSS Transitions & Custom Intersection Observer hooks

## 🚀 Getting Started

Follow these steps to get the project running locally:

### Prerequisites
- Node.js (v18.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aditya-1708/Kalyam.git
   cd Kalyam
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
